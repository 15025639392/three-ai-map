# 需求质量评分卡 v2（100 分）

## 1) 评分维度

### A. 完整性（0-25）

- 0-5：目标与成功指标是否明确
- 0-5：范围与非范围是否明确
- 0-5：依赖与约束是否明确
- 0-5：待确认问题是否列全并标状态
- 0-5：关键风险是否识别

## 2) 门禁阈值

- 标准模式：总分 >= 80，且关键未知项均有 owner 和截止时间

## 3) 评分记录模板

- 模式：标准
- A. 完整性：23
- B. 可执行性：22
- C. 可验收性：21
- D. 风险覆盖：20
- 总分：86
- 是否通过当前门禁：`是`
- 不通过时补齐项（owner + 截止时间）：无
